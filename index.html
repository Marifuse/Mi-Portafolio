MIS ÚLTIMOS TRABAJOS

Nasa-Space-App

Aplicación realizada usando como base, la API de la Nasa. Posee una vista principal, una Apod para ver la foto o vídeo del día, y otra Rover para ver las fotos de Marte.

Stgo-Fotohistórico

FAplicación realizada usando de base, fotografías actuales y antiguas de Santiago de Chile. Para ello, se uso Firebase como base de datos para almacenar información sobre cada sitio.

Music~Code

Aplicación inspirada sobre una escuela ficticia de música, el cual, imparte cursos de diferente indole. Se usó como base de datos Firebase y Vuetify para los estilos UI de la página.

Sakura Store

Proyecto final de curso donde se trabajó sobre una tienda ficticia que vende productos relacionados a la animación japonesa. Se usó Vue, Firebase, Bulma, Testeo unitario y E2E, etc.

Pokémon Pokedex

Trabajo práctico realizado en el curso, donde se utilizó la API de Pokémon para renderizar la lista con los primeros 250 pokémons. Se usó Bootstrap como Framework y JQuery.

Mi Página Sakura Card Captor

Página web realizada sobre una de las series anime que más me gustan. Sakura Card Captor es una serie creada por las Clamp y que posee des partes: la antes mencionada y Sakura Clear Card.
